# car-reservation
A PHP web site - student project (a week of work in december 2022).

Front and back office pages in order to manage car reservations.

&nbsp;

---

## Prerequisites
- A web server
- PHP
- MySQL

### Stack used on this project
- XAMPP v3.3.0
- Apache v2.4.54
- PHP v8.1.10
- MariaDB v10.4.25
